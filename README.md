<center>关于链家全网房价数据分析挖掘项目

**关于该项目**  
本项目是在大数据分析文摘的学习基础之上，结合个人掌握技能和知识进行的时间挖掘实践。
项目部分参考了其他工程师的指导，在此作简单分享，请多指正。

**数据说明**
1. 数据信息：
 - 数据量：40多万
 - 时间：2018年7月前
2. 来源
 - 作者：田昕峣
 - 获取方式：https://github.com/XinyaoTian/lianjia_Spider  

**项目目标**  
建立单位面积房价的预测模型
